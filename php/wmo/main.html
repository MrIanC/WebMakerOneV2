WebMakerOne

Static html Creator

WebMakerOne is a tool designed to simplify the creation of static HTML websites.

Requirements:

  • PHP 8.3
  • Apache/Nginx
  • A MySql Database or equilivant
  • A domain and installed at the web-root. eg: www.example.com or mypage.example.com

Script Installation:

Upload the install.php file to the root of your Web Document Folder. htdocs

From a browser run the install.php from https://www.yourdomain.com/install.php

Follow the instructions to install and setup WebMakerOne

Manual Installation:

Download the latest version of WebMakerOne from Github

Extract the files to the root of your Web Document directory. (htdocs)

In your web document directory there should be a directory named app

Go to https://www.yourdomain.com/app

Follow the instructions to complete the setup WebMakerOne
