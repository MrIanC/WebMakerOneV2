After Installation

Update

The update page is to update WebMakerOne to the latest version. The version control is from the updated date and time.

From the menu select update

Select Update from the Update tab to update the software.

A message log will show the status of the updates

If the Update button is not available, the latest version is already installed or the update details from GitHub are not available.

Secrets

Secrets is to store your API keys. Google Fonts and Gemini

From the menu select secrets

Select from the Secrets tab to save your API Keys.

Get your API keys from the respective websites.

Users

Users is to manage the users of the software.

From the menu select users

To create a user fill in the username and password fields then

Select from the Users tab to add a user.

You can delete a user by clicking the on the right of the username.

You cannot delete the user you are logged in as.
You can update the password by adding a new user with the same username.

Data

Data is to manage the data of your Website.

From the menu select data

Select from the data tab save the site data

This data forms part of the SEO of your website as well as the data for the social media links for your site

Palette

Palette is to manage the colours of your Website.

From the menu select palette

Choose a colour from the colour picker and add a name for the colour

Select Add to add the colour to the palette

Your custom colours are listed on the right and can be deleted using the

The colours are used in the css for the website in the content editor. Once deleted the css will not be applied to the website.
Examples of usage are available at the bottom of the tab. Use these names in the CSS of your website.

Fonts

Fonts is to manage the fonts of your Website.

From the menu select fonts

Choose a font from the list and select the font for the heading, paragraph and navigation

Once selected the fonts will be applied to the website

The fonts are used in the css for the website in the content editor. Once deleted the css will not be applied to the website.
The fonts are from Google Fonts. You will need to add your Google Fonts API key in the Secrets tab.

Includes

Includes is to add scripts, stylesheets or HTML to your Website.

From the menu select includes

Fill in the name, type, url and location of the include

Select Include Link/Script to add the include

The includes are added to the website in the content editor. Once deleted the include will not be applied to the website.
The pallette, fonts and scripts are automatically added to includes

Actions

Actions is to add custom actions to your Website. These are JS or Jquery scripts that will run when an element is clicked.

From the menu select actions

Fill in the name and script of the action

Select Add to add the action

The actions are added to the website in the content editor. Once deleted the action will not be applied to the website.
