Scripts

Scripts is to add custom scripts to your Website. These are JS or Jquery scripts that will run when the page is loaded.

From the menu select scripts

Upload the script using Choose Files then clicking the Upload button

If any fill in the input values for the script then click the button

To enable the script click the Button

To disable the script click the button

To remove the script use the button

The scripts are added to the website and loaded on startup. Once deleted the script will not be applied to the website.
